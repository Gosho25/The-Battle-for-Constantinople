# The Battle for Constantinople
*"From the mud to the gold"*

## Chapter 1: The idea
I am someone who enjoys learning programming and exploring new ideas. My hobbies are programming and playing computer games, so I decided to combine them and create this game. 
I have a big imagination and many ideas — it all started as a joke, but gradually my interest led me to develop the game "The Battle for Constantinople".
I am not a professional, and I know how challenging it is, but I do all of this out of passion. 
My dream is to publish the game so that others can enjoy playing it and immerse themselves in my ideas.
